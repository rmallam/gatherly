# HostEze Images

This folder contains all image assets for the HostEze app.

## Contents

- **Logo Files**: App logos and icons
- **Screenshots**: App screenshots for stores
- **Marketing**: Promotional images

## Image Specifications

### App Icons
- **Android**: 512x512 PNG (for Google Play)
- **Launcher Icons**: Multiple densities (mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi)

### Marketing
- **Social Media**: 1080x1080 (square)
- **Instagram Stories**: 1080x1920 (9:16)
- **Feature Graphic**: 1024x500 (Google Play)

## Usage

Images are referenced from:
- Frontend: `frontend/public/`
- Android: `frontend/android/app/src/main/res/`
- Documentation: Various markdown files

---

**Note**: Keep image files optimized (<50 KB for web, <20 KB for icons)
