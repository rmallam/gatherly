## Step-by-Step Production Guide

### Phase 1: Pre-Production (1 hour)

1. **Write Your Script**
   - Keep it under 30 seconds
   - Focus on one key benefit
   - Include clear CTA

2. **Storyboard**
   - Sketch out 5-6 key scenes
   - Plan transitions
   - Note text overlays

3. **Gather Assets**
   - Record app screen demos
   - Download stock footage
   - Prepare logo/branding

### Phase 2: Recording (2 hours)

#### Record App Demo
1. Clean your phone screen
2. Enable "Do Not Disturb"
3. Use test account with good data
4. Record in good lighting
5. Show key features:
   - Creating an event
   - Adding guests from contacts
   - Generating QR codes
   - Scanning QR code
   - Event wall

#### Generate Voice-Over
1. Go to ElevenLabs
2. Paste your script
3. Choose voice (try different ones)
4. Generate and download

### Phase 3: Editing (3-4 hours)

#### Using CapCut (Recommended for Beginners)

1. **Import Assets**
   - Open CapCut
   - Create new project (9:16 ratio)
   - Import screen recordings
   - Import voice-over

2. **Arrange Clips**
   - Drag voice-over to timeline
   - Cut screen recordings to match voice
   - Trim to 30 seconds total

3. **Add Text Overlays**
   - Use "Text" â†’ "Default"
   - Add key points as text
   - Animate with "In" and "Out" effects
   - Use bold, readable fonts

4. **Add Music**
   - Choose upbeat background music
   - Lower volume to -20dB (voice should be clear)
   - Fade in/out

5. **Add Effects**
   - Transitions between clips (0.5s)
   - Zoom effects on important features
   - Add your logo watermark

6. **Color Grading**
   - Adjust brightness (+10)
   - Increase saturation slightly
   - Add "Film" filter for professional look

7. **Export**
   - Resolution: 1080x1920
   - Frame rate: 30fps
   - Quality: High
   - Format: MP4

### Phase 4: Final Touches

1. **Add Captions**
   - Use CapCut auto-captions
   - Or add manually
   - 80% of Instagram videos watched without sound!

2. **Preview**
   - Watch on phone
   - Check text readability
   - Verify audio levels

3. **Export Multiple Versions**
   - 9:16 for Reels/Stories
   - 1:1 for Feed
   - 16:9 for YouTube (bonus)

## Pro Tips for Instagram Ads

### Visual Best Practices
1. **First 3 seconds are crucial** - Hook immediately
2. **Use bright colors** - Stand out in feed
3. **Show faces** - Increases engagement by 38%
4. **Add captions** - 85% watch without sound
5. **Keep it simple** - One message per video

### Text Overlays
- **Font**: Bold, sans-serif (Arial, Helvetica)
- **Size**: Large enough to read on mobile
- **Color**: High contrast (white text on dark bg)
- **Animation**: Subtle, not distracting

### Music Selection
- **Upbeat tempo** - 120-140 BPM
- **Trending sounds** - Check Instagram trends
- **Volume**: Background only (-20dB to -15dB)

### Call to Action
- **Clear and direct**: "Download now"
- **Create urgency**: "Limited time free"
- **Show benefit**: "Start planning today"
